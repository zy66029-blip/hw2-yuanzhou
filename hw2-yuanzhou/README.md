# Week 2 GenAI Workflow Assignment

## Workflow
Drafting customer support responses

## User
A small business customer support staff member

## Input
A customer email or message describing a question, complaint, or request

## Output
A professional draft response email in English

## Why this workflow is valuable
Customer support teams spend a lot of time writing repetitive replies. 
A draft-generation workflow can save time, improve tone consistency, and help staff respond faster. 
However, sensitive cases may still require human review.

## How to Run
1. Install Python
2. Install dependency:
   pip install google-genai
3. Set API key:
   export GEMINI_API_KEY="your_api_key_here"
4. Run:
   python3 app.py
